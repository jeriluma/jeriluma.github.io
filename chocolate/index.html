Chocolate Infographic | Project Notes

UW | INFO 424: Information Visualizations | Mar 2014 - June 2014

  • UX / UI

Description

An individual project to design an infographic by analyzing and learning from other infographics. I chose chocolate as my infographic topic, and explored how to communicate information about chocolate through numbers, location, appearance, relationship, time and layout.

To inspire us to create our infograhpics, we utilized models to:

  1. 1. develop analyical skills why these infograhpics are successful
  2. 2. understand how those infographics are successful
  3. 3. be able to implement what we learn

Methods Used

  • Analyzing designs
  • Critique and Review
  • Communicating in writing and video

Tools Used

Adobe Illustrator
